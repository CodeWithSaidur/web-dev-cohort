String properties and methods

1.1. .length              Returns the number of characters in a string
1.2. .toUpperCase() 	    Converts to uppercase
1.3. .toLowerCase()	    Converts to lowercase
1.4. .indexOf()           Returns the first occurrence index of a substring
1.5. .lastIndexOf()	    Returns the last occurrence index
1.6. .includes()	        Checks if a substring exists
1.7. .startsWith()	    Checks if string starts with a given substring
1.1.8. .endsWith()	        Checks if string ends with a given substring
1.1.9. slice(start, end)	    Extracts a part of the string
1.10..substring(start, end)	Similar to slice(), but doesn't accept negative values
1.11. . .trim()
1.12. . .replace('World','Sabed')
1.13. . .replaceAll('World',"Sabed")
1.14. .charAt(6)
1.15. .charCodeAt(6)
1.16. .at(-5)
